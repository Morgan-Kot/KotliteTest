# **Kotlite Launcher**

> *Lite News: Launcher*

---

### **What is Kotlite Launcher?**
The **Kotlite Launcher** is the home of all **Kotlite Browser** releases.  
Instead of checking this GitHub repo every time, you can install the Launcher and download the Kotlite Browser version you want — all in one place.

---

### **Why use Kotlite Launcher?**
The **Kotlite Launcher** makes it easier to install and launch different **Kotlite Browser** releases.  
Instead of logging into GitHub to see if a new version is out, you can simply open the Launcher and download the latest release.

The Launcher also makes installing and deleting Kotlite Browser versions much easier.

---

### **Is it free to use?**
YES! The Kotlite Launcher is 100% free to use.  
No payment. Ever.

---

### **Will it get updates?**
Yes. We will continue to update and improve both the Launcher and the Browser.

---
