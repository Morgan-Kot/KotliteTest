# Kotlite Launcher

> *Lite News: Launcher*
#

### What is *Kotlite Launcher?*
The Kotlite Launcher is the home of all the **kotlite browser** [releases](https://github.com/Morgan-Kot/KotliteBrowser/releases).
Instead of checking this **Github repo** you can simply install the Kotlite Launcher and download the Kotlite Browser you want.

#

### Why use *Kotlite Launcher?*
The **Kotlite Launcher** makes it easier to install / launch different **Kotlite Browser** [releases](https://github.com/Morgan-Kot/KotliteBrowser/releases).
Instead of logging into [Github](https://github.com) to check if a new version has arrived, you can now simply login to **Kotlite Browser** and download the latest
[release](https://github.com/Morgan-Kot/KotliteBrowser/releases)
**Kotlite Browser** also makes installing and deleting Kotlite Browser versions easier
