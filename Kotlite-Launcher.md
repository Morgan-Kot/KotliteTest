# Kotlite Launcher

> *Lite News: Launcher*
#

### What is *Kotlite Launcher?:*
The Kotlite Launcher is the home of all the **kotlite browser** releases.
Instead of checking this **Github repo** you can simply install the Kotlite Launcher and download the Kotlite Browser you want.

#

### Why use *Kotlite Launcher*
The **Kotlite Launcher** makes it easier to install / launch different **Kotlite Browser** [releases](https://link-url-here.org)
